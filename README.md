# Blueprint Skill

Methodology plugin for Claude Code. Battle-tested on 5+ production projects.

## Philosophy

> "Le cerveau planifie, les mains executent."
> Opus thinks. Sonnet builds. Opus verifies. One shot. No iteration in front of the user.

## Skills

- **blueprint-plan** — Produce an ultra-detailed implementation blueprint (CDC) before writing any code. Compare 3+ approaches. Every step copy-pasteable. Zero ambiguity.
- **blueprint-execute** — Execute a blueprint mechanically, step by step. No improvisation. No deviation. Stop on error.
- **challenge-assumptions** — Challenge your own approach before committing. Compare 3+ options. Kill cognitive biases. Pick the best, not the first.
- **verification-proof** — Prove that work is done with concrete evidence. The word "should" is banned. EXISTS → SUBSTANTIVE → WIRED.

## Agents

- **blueprint-reviewer** — Reviews completed work against the original blueprint. Detects every deviation.

## Commands

- **/blueprint** — Full workflow: plan → execute → verify → review

## Origin

Forged from the SOUL.md of Saphir (personal assistant bot) and the CLAUDE.md of a production VPS managing 9 Docker services. Every rule exists because we learned it the hard way.
